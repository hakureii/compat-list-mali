## What the statuses mean

* ``Nothing`` - Opens, no logs created
* ``Boots`` - Game opens, usually doesn't progress far
* ``In-game`` - Not reported as being finishable, usually has bugs
* ``Playable`` - Reported as finishable
* ``Perfect`` - No bugs
